Homelab Support

Homelab is a dashboard app for self-hosted services. If you need help, use the resources below.

Get Help

Report a bug or request help on GitHub Issues View the Homelab project repository

Before Opening An Issue

  • Confirm you are using the latest version of Homelab.
  • Check that your service URL is reachable from your device.
  • Verify your API key, token, username, password, or two-factor code.
  • Include the service type, app version, platform, and a short description of the problem.

Privacy

Read the Homelab Privacy Policy for information about local storage, credentials, and network connections.
